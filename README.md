<p>To run the app locally : 
<ul>
	<li>clone the app</li>
	<li>cd to app root</li>
	<li>run composer install</li>
	<li>edit database credential in .env</li>
	<li>run php artisan migrate</li>
	<li>run php artisan db:seed</li>
	<li>run php artisan serve</li>
</ul>
</p>
